# Gaofen Series Data

## GID

The data could be downloaded from [here](https://x-ytong.github.io/project/GID.html).

Please run the following command to re-organize the dataset.

```shell
python tools/dataset_converters/gid.py --data-root /path/to/GID --out-root /path/to/Save
```
